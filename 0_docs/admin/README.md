# Раздел «Администрирование» (контент)

- **`articles/`** — статьи с префиксами `01–10` для порядка в дереве репозитория. Публичный путь к странице на сайте задаётся движком (обычно совпадает с путём к `.md` без расширения).
- **`images/`** — иллюстрации по подпапкам на статью (`organizacii`, `pak`, …). В markdown пути вида `/content/0_docs/admin/images/<статья>/<файл>.png`.
- **`images/_unused/`** — PNG, которые не встречались в статьях (оставлены на случай доразметки).

После этой раскладки каталог **`images/admin/`** в корне репозитория **не используется** для админки.
