# MedControl — контент документации

Локальный репозиторий со статьями (Markdown), изображениями и служебными файлами для сайта документации.

Структура корня повторяет каталог `docs-app/public/content/` в движке: сюда кладётся то, что при сборке должно оказаться в `public/content/`.

## Связь с движком

Пока движок (`medcontrol-docs-platform`) использует свою копию `public/content/`. Этот репозиторий — отдельная точка правки контента; позже его подключат к движку (submodule или шаг CI).

## Базовая линия

Снимок проверок ссылок и списка статей до разделения репозиториев зафиксирован в движке: файл `baseline-links-before-migration.txt`.
